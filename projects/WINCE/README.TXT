
README.TXT for the Windows CE VS Projects
--------------------------------------------------------------------------------

The VS projects are:

appweb-dynamic - Builds Appweb using dynamic linking. All modules are loadable.
appweb-static  - Builds Appweb using static linking. All modules are builtin.
