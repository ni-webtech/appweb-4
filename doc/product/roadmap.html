Appweb Roadmap

The Appweb roadmap is created by listening to user input. As such, it is subject to change. This roadmap is not a commitment to product, but the current thinking about future directions.

Those who shout loudest -- often get what they ask for. To request a feature, please send email to:

dev@embedthis.com

What's in the Next Release

The next release in the stable branch will be the Embedthis Appweb 4.0.1. Patches to the stable branch are typically released as issues require.

The next release will be Appweb 4.0 and it brings some exciting new functionality.

  • Modular Ejscript supporting CommonJS specifications
  • Improved documentation and samples
  • Small footprint Ejscript
  • Async I/O and enhanced streaming support

For Appweb 3, is now regarded as legacy code. Customers should upgrade to Appweb 3.

Please tell us what you would like to see at dev@embedthis.com.
